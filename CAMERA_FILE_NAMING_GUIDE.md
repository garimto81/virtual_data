# 📹 카메라 파일명 규칙 가이드

## 🎯 파일명 구조

### Google Sheets 저장값 vs 실제 파일명

| 구분 | 시트 저장값 | 실제 파일명 | 설명 |
|------|------------|-------------|------|
| **시트 입력** | `1` | `CamAa0001.mp4` | 시트에는 간단한 번호만 저장 |
| **시트 입력** | `2` | `CamAa0002.mp4` | 실제 파일명은 카메라명+4자리 번호+확장자 |
| **시트 입력** | `10` | `CamAa0010.mp4` | 자동으로 4자리로 패딩 |
| **시트 입력** | `100` | `CamAa0100.mp4` | 세자리 숫자도 4자리로 변환 |
| **시트 입력** | `1000` | `CamAa1000.mp4` | 네자리는 그대로 유지 |

## 📋 파일명 생성 규칙

### 1. 카메라 이름 추출
- **Type 시트 A2**: 첫 번째 카메라 이름 (예: "CamAa")
- **Type 시트 A3**: 두 번째 카메라 이름 (예: "CamBb")

### 2. 번호 할당 방식
- **시작 번호**: 1 (설정 가능)
- **증가 간격**: 1 (설정 가능)
- **번호 진행**: 1 → 2 → 4 → 5 → 7 → 8 ...
  - Cam1: 홀수 (1, 3, 5, 7, 9...)
  - Cam2: 짝수 (2, 4, 6, 8, 10...)

### 3. 최종 파일명 형식
```
[카메라이름][4자리번호].mp4
```

## 🔄 변환 과정 예시

### Type 시트 설정
```
A2: CamAa
A3: CamBb
```

## 📊 Index 시트 구조

| 열 | 내용 | 예시 | 설명 |
|----|------|------|------|
| **J열** | cam | `CamAa+CamBb` | 카메라 조합 |
| **K열** | camFile01name | `CamAa` | 첫 번째 카메라 이름 |
| **L열** | camFile01number | `1` | 첫 번째 카메라 번호 (간단) |
| **M열** | camFile02name | `CamBb` | 두 번째 카메라 이름 |
| **N열** | camFile02number | `2` | 두 번째 카메라 번호 (간단) |

## 🎬 실제 사용 시나리오

### 영상 파일 검색 시
1. **시트에서 확인**: HandNumber H001 → Cam1no: `1`, Cam2no: `2`
2. **파일명 생성**: 
   - `CamAa` + `0001` + `.mp4` = `CamAa0001.mp4`
   - `CamBb` + `0002` + `.mp4` = `CamBb0002.mp4`
3. **파일 시스템에서 검색**: 생성된 파일명으로 동영상 파일 찾기

### 장점
- **시트 간결성**: 번호만 저장하여 가독성 향상
- **유연성**: 카메라 이름 변경 시 Type 시트만 수정
- **확장성**: 새로운 카메라 추가 시 쉽게 대응
- **일관성**: 모든 파일명이 동일한 규칙으로 생성

## 🔧 문제 해결

### Q: 파일명에 특수문자가 포함된 경우?
**A**: Type 시트의 카메라 이름을 파일 시스템에 안전한 이름으로 변경

### Q: 번호가 4자리를 초과하는 경우?
**A**: 자동으로 필요한 자릿수만큼 확장 (예: 10000 → 10000)

### Q: 중복 번호 방지는?
**A**: 순차적으로 할당하므로 중복 없음. 기존에 카메라 정보가 있는 행은 건너뜀
